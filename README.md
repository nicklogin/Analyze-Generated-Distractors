# Analyze-Generated-Distractors

## Описание репозитория

Данный репозиторий содержит материалы и данные исследования, описанного в Разделе 3.2 "Анализ сгенерированных дистракторов и сравнение различных методов" диссертации Логина Никиты Вячеславовича <b>"Автоматизация создания неправильных вариантов (дистракторов) для банков заданий языкового тестирования"</b> на соискание учёной степени кандидата филологических наук.

## Структура репозитория

Папка "synt-complexity" содержит данные и исходный код экспериментов по анализу дистракторов, полученных разными методами, с точки зрения синтаксической сложности.

Папка "general" содержит данные и исходный код остальных описанных в Разделе 3.2 экспериментов.

## Папка "general"

### Структура папки

- Подпапка "notebooks" содержит интерактивные блокноты Jupyter с исходным кодом проведённых экспериментов и исследований
- Подпапка "data_for_comparison" содержит данные, используемые в аналитических сравнениях (корреляциях, кросс-тестах)
- Подпапка "data_input" содержит входные данные для интерактивных блокнотов
- Подпапка "data_output_image" содержит изображения, полученные в результате работы интерактивных блокнотов
- Подпапка "data_output_table" содержит таблицы, полученные в результате работы интерактивных блокнотов
- Подпапка "tools" содержит вспомогательные модули Python, используемые в интерактивных блокнотах

| Файл             | Описание           |
|------------------|--------------------|

| cross_test_ege.ipynb | Кросс-тест лексических метрик на данных ЕГЭ |
| cross_test_ru_race_tf.ipynb | Кросс-тест лексических метрик на данных Ru-RACE-TF |
| get_metrics_table.ipynb | Формирование таблицы со значениями предлагаемых метрик |

| relevance_metrics.ipynb | Вычисление предлагаемых метрик релевантности |
| score_outputs_gemini | Вычисление метрики "неправильности" дистракторов при помощи модели Gemini |
